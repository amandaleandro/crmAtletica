# Wireframes – CRM Atlética Universitária (MVP)

## Web App (Desktop)

- Login
- Dashboard (cards: membros, eventos, financeiro, patrocínios)
- Membros (lista, busca, filtro, perfil detalhado)
- Modalidades/Times (lista, elenco, treinos)
- Eventos (lista, criar, inscrições, check-in)
- Financeiro (pendências, lançamentos, caixa)
- Patrocínios (pipeline Kanban)

## Mobile-first (PWA ou app)

- Login
- Dashboard resumido (meus eventos, pagamentos, treinos)
- Meus Dados (perfil, pagamentos)
- Eventos (lista, inscrição, check-in QR)
- Treinos (lista, check-in)
- Notificações (mensagens, alertas)

---

## Estrutura de Telas (resumida)

### Login
- Email, senha, botão entrar

### Dashboard
- Cards: membros ativos, eventos próximos, saldo caixa, leads de patrocínio

### Membros
- Lista com busca/filtro
- Perfil: dados, times, pagamentos, documentos

### Modalidades/Times
- Lista de modalidades
- Elenco do time, calendário de treinos

### Eventos
- Lista de eventos
- Página do evento: detalhes, inscritos, check-in

### Financeiro
- Lista de pendências
- Lançar entrada/saída
- Extrato do caixa

### Patrocínios
- Pipeline Kanban: lead, proposta, negociação, fechado, perdido

### Mobile
- Telas simplificadas, foco em check-in, inscrições, pagamentos rápidos
